Privacy Policy

We take your privacy seriously. Here's what you need to know about our extension:

  • Data Collection: We do not collect, store, or transmit any personal information or browsing history.
  • Translation Service: We use Google Translate's public API for translations. Only the subtitle text is sent for translation.
  • Storage: We only store your language preferences and settings locally in your browser.
  • Permissions: We request only the minimum permissions needed:
    • activeTab: To access subtitle elements on Bilibili
    • storage: To save your preferences
    • scripting: To inject translated subtitles
    • bilibili.com host permission: To function on Bilibili videos
  • Third-Party Services: We only interact with Google Translate's API for translation purposes.
  • Updates: All code updates are delivered through the Chrome Web Store.

Contact

For privacy concerns or questions, please open an issue on our GitHub repository.

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any changes by updating the version number in the Chrome Web Store.
